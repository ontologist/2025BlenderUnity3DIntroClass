Introduction / 紹介

Cursor is an AI-powered code editor built on VS Code that will help you write and understand code for Unity3D. It provides intelligent code completion, explanations, and assistance - perfect for learning game development!

Key Features for This Course:

  • AI-powered code completion and suggestions
  • Natural language code explanations
  • Built-in terminal for Git commands
  • C# language support for Unity scripting
  • GitHub integration

CursorはVS Codeをベースにした AI搭載のコードエディタで、Unity3D用のコードを書き、理解するのに役立ちます。インテリジェントなコード補完、説明、アシスタンスを提供し、ゲーム開発の学習に最適です！

このコースの主な機能:

  • AI搭載のコード補完と提案
  • 自然言語によるコード説明
  • Gitコマンド用の組み込みターミナル
  • UnityスクリプティングのためのC#言語サポート
  • GitHub統合

Step 1: Download Cursor / ステップ1：Cursorをダウンロード

  1. Visit the Cursor website: Go to https://www.cursor.com
  2. Download for your operating system:
    • Windows: Click "Download for Windows"
    • Mac: Click "Download for Mac"
      • For Apple Silicon (M1/M2/M3): Download ARM version
      • For Intel Macs: Download Intel version
    • Linux: Click "Download for Linux"
  3. Wait for download to complete (File size is approximately 100-150 MB)
  1. Cursorウェブサイトにアクセス: https://www.cursor.com にアクセス
  2. オペレーティングシステム用にダウンロード:
    • Windows: 「Download for Windows」をクリック
    • Mac: 「Download for Mac」をクリック
      • Apple Silicon（M1/M2/M3）の場合：ARMバージョンをダウンロード
      • Intel Macの場合：Intelバージョンをダウンロード
    • Linux: 「Download for Linux」をクリック
  3. ダウンロードが完了するまで待つ（ファイルサイズは約100-150 MB）

Step 2: Install Cursor / ステップ2：Cursorをインストール

For Windows / Windows用

  1. Locate the downloaded file: CursorSetup.exe
  2. Double-click to run the installer
  3. If Windows SmartScreen appears, click "More info" → "Run anyway"
  4. Follow the installation wizard
  5. Click "Install" and wait 1-2 minutes
  6. Click "Finish" to launch Cursor
  1. ダウンロードしたファイルを見つける：CursorSetup.exe
  2. ダブルクリックしてインストーラーを実行
  3. Windows SmartScreenが表示された場合、「詳細情報」→「実行」をクリック
  4. インストールウィザードに従う
  5. 「インストール」をクリックして1〜2分待つ
  6. 「完了」をクリックしてCursorを起動

For Mac / Mac用

  1. Locate the downloaded file: Cursor.dmg
  2. Double-click to open the disk image
  3. Drag the Cursor icon to the Applications folder
  4. Eject the Cursor disk image
  5. Open Applications folder and double-click Cursor to launch
  6. If macOS security warning appears, right-click Cursor → "Open"
  1. ダウンロードしたファイルを見つける：Cursor.dmg
  2. ダブルクリックしてディスクイメージを開く
  3. CursorアイコンをApplicationsフォルダにドラッグ
  4. Cursorディスクイメージを取り出す
  5. Applicationsフォルダを開いてCursorをダブルクリックして起動
  6. macOSセキュリティ警告が表示された場合、Cursorを右クリック →「開く」

Step 4: Install Essential Extensions / ステップ4：必須の拡張機能をインストール

For Unity development, install these extensions:

  1. Open Extensions View: Press Ctrl+Shift+X (Windows/Linux) or Cmd+Shift+X (Mac)
  2. Install C# Extension: Search for "C# Dev Kit" by Microsoft → Click "Install"
  3. Install Unity Code Snippets (Optional): Search for "Unity Code Snippets" → Click "Install"
  4. Install GitLens (Optional): Search for "GitLens" → Click "Install"

Unity開発には以下の拡張機能をインストール:

  1. 拡張機能ビューを開く: Ctrl+Shift+X（Windows/Linux）またはCmd+Shift+X（Mac）を押す
  2. C#拡張機能をインストール: Microsoftの「C# Dev Kit」を検索 → 「Install」をクリック
  3. Unity Code Snippetsをインストール（オプション）: 「Unity Code Snippets」を検索 → 「Install」をクリック
  4. GitLensをインストール（オプション）: 「GitLens」を検索 → 「Install」をクリック

Step 5: Connect to GitHub / ステップ5：GitHubに接続

  1. Click the Source Control icon in the left sidebar (or press Ctrl+Shift+G / Cmd+Shift+G)
  2. Click "Sign in with GitHub"
  3. A browser window will open - click "Authorize"
  4. Close the browser and return to Cursor
  5. You can now commit and push directly from Cursor!
  1. 左サイドバーのソースコントロールアイコンをクリック（またはCtrl+Shift+G / Cmd+Shift+Gを押す）
  2. 「Sign in with GitHub」をクリック
  3. ブラウザウィンドウが開きます - 「Authorize」をクリック
  4. ブラウザを閉じてCursorに戻る
  5. これでCursorから直接コミットしてプッシュできます！

Using Cursor's AI Features / CursorのAI機能を使う

1. Cursor Chat (Ctrl+L or Cmd+L)

Ask questions about code in natural language:

  • "What does this Unity script do?"
  • "How do I move a GameObject with arrow keys?"
  • "Explain what a Rigidbody is in Unity"

2. AI Code Completion (Automatic)

As you type, Cursor suggests code completions. Press Tab to accept.

3. Code Explanation (Ctrl+K or Cmd+K)

Select code and press Ctrl+K / Cmd+K, then type:

  • "explain" - Get code explanation
  • "comment" - Add comments
  • "fix" - Find and fix errors

1. Cursor Chat（Ctrl+LまたはCmd+L）

自然言語でコードについて質問:

  • 「このUnityスクリプトは何をしますか？」
  • 「矢印キーでGameObjectを移動するにはどうすればよいですか？」
  • 「UnityでRigidbodyとは何か説明してください」

2. AIコード補完（自動）

入力すると、Cursorがコード補完を提案。Tabを押して受け入れ。

3. コード説明（Ctrl+KまたはCmd+K）

コードを選択してCtrl+K / Cmd+Kを押し、次のように入力:

  • 「explain」- コード説明を取得
  • 「comment」- コメントを追加
  • 「fix」- エラーを見つけて修正

Troubleshooting / トラブルシューティング

Problem: Cursor won't start

  • Solution 1: Restart your computer
  • Solution 2: Reinstall Cursor
  • Solution 3: Check system requirements (8GB RAM minimum)

Problem: AI features not working

  • Solution 1: Make sure you're signed in (check bottom left corner)
  • Solution 2: Check internet connection
  • Solution 3: You may have reached daily limit (free plan)

問題：Cursorが起動しない

  • 解決策1: コンピュータを再起動
  • 解決策2: Cursorを再インストール
  • 解決策3: システム要件を確認（最小8GB RAM）

問題：AI機能が動作しない

  • 解決策1: サインインしていることを確認（左下隅をチェック）
  • 解決策2: インターネット接続を確認
  • 解決策3: 1日の制限に達している可能性（無料プラン）
