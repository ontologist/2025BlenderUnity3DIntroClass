Assignment Description / 課題概要

English: Enhance your Week 1 maze by adding materials, textures, and a simple path animation. This assignment builds on your existing maze project and demonstrates your understanding of materials, UV unwrapping, texture application, and basic animation.

日本語: 第1週の迷路にマテリアル、テクスチャ、シンプルなパスアニメーションを追加して強化します。この課題は既存の迷路プロジェクトをベースにして、マテリアル、UV展開、テクスチャ適用、基本的なアニメーションの理解を示します。

Requirements / 要件

1. Materials Application 5 points

English:

  • Apply materials to ALL objects in your maze
  • Use appropriate colors for different elements:
    • Floor: Green/brown (grass or dirt feel)
    • Walls: Gray (stone-like)
    • Start Area: Blue or green
    • Goal: Bright red or gold (eye-catching)
  • Adjust material properties (Metallic and Roughness) appropriately
  • Name your materials descriptively (e.g., "Wall_Stone", "Floor_Grass")

日本語:

  • 迷路のすべてのオブジェクトにマテリアルを適用
  • 異なる要素に適切な色を使用：
    • 床: 緑/茶色（草または土の感じ）
    • 壁: 灰色（石のような）
    • スタートエリア: 青または緑
    • ゴール: 明るい赤または金色（目を引く）
  • マテリアルプロパティ（MetallicとRoughness）を適切に調整
  • マテリアルを説明的に命名（例：「Wall_Stone」、「Floor_Grass」）

2. UV Unwrapping & Texture Application 5 points

English:

  • UV unwrap at least ONE object (wall segment recommended)
  • Apply at least ONE image texture to the unwrapped object
  • Texture requirements:
    • Use appropriate seam placement
    • Texture should not appear overly stretched or distorted
    • Recommended: brick, stone, or wood texture for walls
    • Image size: at least 512x512 pixels
  • Texture file must be saved in project folder
  • Use File → External Data → Pack Resources to embed textures

日本語:

  • 少なくとも1つのオブジェクトをUV展開（壁セグメント推奨）
  • 展開したオブジェクトに少なくとも1つの画像テクスチャを適用
  • テクスチャ要件：
    • 適切なシーム配置を使用
    • テクスチャが過度に引き伸ばされたり歪んだりしていないこと
    • 推奨：壁用のレンガ、石、または木材テクスチャ
    • 画像サイズ：少なくとも512x512ピクセル
  • テクスチャファイルはプロジェクトフォルダに保存する必要があります
  • File → External Data → Pack Resourcesを使用してテクスチャを埋め込む

🔗 Free Texture Resources:

  • Poly Haven - High-quality PBR textures
  • OpenGameArt.org - Game assets
  • Textures.com - Limited free downloads
  • CC0 Textures - Public domain

3. Path Animation 3 points

English:

  • Create ONE simple path animation in your maze
  • Animation options (choose one):
    • A sphere or object moving along a path through the maze
    • A "collectible" item rotating or moving in place
    • A decorative element (e.g., floating gem, patrol guard)
  • Animation requirements:
    • Duration: 3-5 seconds (90-150 frames at 30fps)
    • Use Follow Path constraint
    • Object should follow path smoothly
    • Path should be visible in the Blender file (don't hide the curve)

日本語:

  • 迷路に1つのシンプルなパスアニメーションを作成
  • アニメーションオプション（1つ選択）：
    • 迷路を通るパスに沿って移動する球体またはオブジェクト
    • 回転または所定の位置で移動する「収集可能」アイテム
    • 装飾要素（例：浮遊する宝石、パトロール警備員）
  • アニメーション要件：
    • 期間：3〜5秒（30fpsで90〜150フレーム）
    • Follow Path制約を使用
    • オブジェクトはパスに滑らかに従う
    • パスはBlenderファイルで表示されている（曲線を非表示にしない）

Step-by-Step Guide / ステップバイステップガイド

Quick Steps / クイックステップ

  1. Open Week 1 Maze
    • Open your week01_maze_[yourname].blend
    • Save As: week-02/week02_maze_[yourname].blend
  2. Apply Materials (30 minutes)
    • Switch to Shading workspace
    • Select floor → Add material → Set green/brown color
    • Select walls → Add material → Set gray color
    • Select goal → Add material → Set red/gold + adjust Metallic/Roughness
    • Rename all materials descriptively
  3. Add Texture (45 minutes)
    • Download a brick or stone texture (512x512+)
    • Save to week-02/textures/ folder
    • Select one wall segment
    • Tab → Edit Mode
    • Mark seams: select vertical edges → Ctrl+E → Mark Seam
    • Select All (A) → U → Unwrap
    • In Shader Editor: Add → Texture → Image Texture
    • Open your texture file
    • Connect Image Texture Color to Principled BSDF Base Color
  4. Create Animation (30 minutes)
    • Shift+A → Curve → Bezier Curve
    • Tab → Edit Mode → Shape path through maze
    • Shift+A → Mesh → UV Sphere (animated object)
    • Select sphere → Constraints → Add → Follow Path
    • Target: Select your curve
    • Check "Follow Curve"
    • Select curve → Object Data Properties → Path Animation
    • Set Frames: 0 to 120
    • Press Spacebar to test
  5. Screenshot & Finalize
    • NumPad 7 (top view)
    • Home key (frame all)
    • Render → Viewport Render Image
    • Save as: week02_maze_screenshot.png
    • File → External Data → Pack Resources
    • Save: Ctrl+S
  6. Submit to GitHub
    • Open GitHub Desktop
    • Commit: "Complete Week 2 maze assignment"
    • Push to GitHub
    • Verify on GitHub.com

Common Mistakes / よくある間違い

❌ Avoid These Mistakes:

  • Materials not visible: Check viewport shading mode (use Material Preview or Rendered)
  • Stretched textures: UV unwrap not done or seams placed incorrectly
  • Missing textures: Forgot to pack resources or textures not in project folder
  • Animation not playing: Follow Path constraint target not set or frame range incorrect
  • Textures too small: Use at least 512x512 pixel images
  • Not saving in correct folder: Must be in week-02/ folder in your GitHub repo
  • Forgetting to push: Committed locally but didn't push to GitHub

FAQ / よくある質問

Q1: Do I need to texture everything?

A: No, only ONE object needs UV unwrapping and texture. Others can use solid color materials.

Q2: Can I use photos as textures?

A: Yes! Photos of walls, floors, etc. work great. Best if taken straight-on with good lighting.

Q3: My animation isn't smooth. What's wrong?

A: Check that: (1) Follow Curve is enabled, (2) Frame range is set correctly, (3) Curve has enough control points

Q4: Can I start from scratch instead of using Week 1 maze?

A: You can, but it's better to build on your existing work. This mirrors real game development.

Q5: How long should this take?

A: Approximately 2-3 hours for most students.
