Segment 2: Introduction to Materials

What are Materials? / マテリアルとは何か？

English: Materials define how surfaces appear in 3D space. They control properties like color, shininess, transparency, and how light interacts with the surface.

日本語: マテリアルは3D空間で表面がどのように見えるかを定義します。色、光沢、透明度、そして光が表面とどのように相互作用するかを制御します。

The Rendering Equation: Materials + Lighting = Final Appearance

レンダリング方程式: マテリアル + ライティング = 最終的な見た目

Material Properties / マテリアルプロパティ

  • Base Color: The fundamental color of the surface / 表面の基本色
  • Metallic: 0 = non-metal (plastic, wood), 1 = metal (steel, gold) / 0 = 非金属（プラスチック、木）、1 = 金属（鋼、金）
  • Roughness: 0 = mirror-like, 1 = completely matte / 0 = 鏡のよう、1 = 完全にマット
  • Alpha: Transparency level / 透明度レベル

Shader Editor Overview / Shader Editor概要

English:

  • Access via Shading workspace (top menu tabs)
  • Node-based system: boxes connected by lines
  • Principled BSDF: Industry-standard shader with all common properties
  • Material Output: Final connection to render engine

日本語:

  • Shadingワークスペース経由でアクセス（上部メニュータブ）
  • ノードベースシステム：線で接続されたボックス
  • Principled BSDF: すべての一般的なプロパティを持つ業界標準シェーダー
  • Material Output: レンダーエンジンへの最終接続

🎹 Quick Access / クイックアクセス:

  • Z → Change viewport shading mode / ビューポートシェーディングモードを変更
  • Choose Material Preview (3rd sphere) to see materials / マテリアルを見るにはMaterial Preview（3番目の球体）を選択
  • Choose Rendered (4th sphere) for full lighting / 完全なライティングにはRendered（4番目の球体）を選択

Segment 3: Applying Materials

Step-by-Step Process / ステップバイステッププロセス

1. Select Object / オブジェクトを選択

Click object in viewport or outliner
ビューポートまたはアウトライナーでオブジェクトをクリック

2. Add Material / マテリアルを追加

  • Switch to Shading workspace (top menu)
  • Properties panel → Material Properties (sphere icon)
  • Click + New to add material

3. Adjust Properties / プロパティを調整

  • Click on Base Color → Choose color from color picker
  • Adjust Metallic slider (0 = plastic/wood, 1 = metal)
  • Adjust Roughness slider (0 = shiny mirror, 1 = matte)

4. See Results / 結果を確認

  • Change viewport shading to Material Preview (3rd sphere icon, top right)
  • Or use Rendered view (4th sphere icon) for full lighting

Material Management / マテリアル管理

  • Naming: Double-click material name to rename descriptively (e.g., "Wall_Stone", "Floor_Grass")
  • Reusing: Select object → Material Properties → click dropdown → choose existing material
  • Multiple Materials: One object can have multiple material slots for different parts

  • Walls: Gray stone - Roughness: 0.6
  • Goal: Bright red or gold - Metallic: 0.5, Roughness: 0.3
  • Start area: Blue - Roughness: 0.7

  • 床：緑または茶色（草/土の感じ） - ラフネス：0.8

  • スタートエリア：青 - ラフネス：0.7

Segment 4: Introduction to UV Unwrapping and Textures

What is UV Unwrapping? / UV展開とは何か？

English: UV unwrapping is the process of "unfolding" a 3D model into a 2D layout. Think of it like unwrapping a gift box and laying the wrapping paper flat.

日本語: UV展開は、3Dモデルを2Dレイアウトに「展開」するプロセスです。ギフトボックスを開いて包装紙を平らにするようなものと考えてください。

  • UV = 2D coordinates: U and V axes (like X and Y, but for textures)
  • Why? To apply 2D images (textures) onto 3D surfaces accurately
  • UV Map: Tells Blender which part of the image goes on which part of the model

UV Unwrapping Basics / UV展開の基礎

Steps / 手順:
1. Enter Edit Mode (Tab)
   編集モードに入る（Tab）

2. Select All Faces (A)
   すべての面を選択（A）

3. Mark Seams (Ctrl+E → Mark Seam)
   シームをマーク（Ctrl+E → Mark Seam）
   - Seams = "cuts" where model will unfold
   - シーム = モデルが展開される「カット」

4. Unwrap (U → Unwrap)
   展開（U → Unwrap）

5. View UV Layout
   UVレイアウトを表示
   - Switch to UV Editing workspace
   - UV Editingワークスペースに切り替え

🔍 Where to Place Seams / シームを配置する場所:

  • Sharp corners and edges / 鋭い角とエッジ
  • Hidden areas (bottom of objects) / 隠れた領域（オブジェクトの底）
  • Natural breaks (where materials change) / 自然な切れ目（マテリアルが変わる場所）

Tip: Fewer seams = smoother textures, but harder to unwrap complex shapes

ヒント: シームが少ない = 滑らかなテクスチャ、しかし複雑な形状の展開は難しい

Applying Image Textures / 画像テクスチャの適用

1. Get Texture Image / テクスチャ画像を取得

  • Free sources: Poly Haven, OpenGameArt.org, Textures.com
  • Or create simple texture in image editor (GIMP, Photoshop, Paint.NET)
  • Recommended size: 512x512 or 1024x1024 pixels

2. Add Texture to Material / マテリアルにテクスチャを追加

In Shader Editor / Shader Editorで:
1. Add → Texture → Image Texture
   Add → Texture → Image Texture
2. Click "Open" and select image file
   「Open」をクリックして画像ファイルを選択
3. Drag from Image Texture "Color" output to Principled BSDF "Base Color" input
   Image Textureの「Color」出力からPrincipled BSDFの「Base Color」入力にドラッグ

3. Adjust UVs if Needed / 必要に応じてUVを調整

  • Switch to UV Editing workspace
  • Select faces in Edit Mode
  • In UV Editor: S to scale, R to rotate, G to move

インストラクターが1つの壁セグメントにレンガテクスチャを適用するデモに従ってください。

Segment 5: Path-Based Animation

What is Animation? / アニメーションとは何か？

English: Animation creates the illusion of movement by displaying a sequence of images (frames) over time. In games and 3D graphics, we typically use 24-30 frames per second.

日本語: アニメーションは、時間をかけて画像（フレーム）のシーケンスを表示することで動きの錯覚を作り出します。ゲームと3Dグラフィックスでは、通常24〜30フレーム/秒を使用します。

Path Animation: Object follows a predefined curve or path / オブジェクトが事前定義された曲線またはパスに従う

Creating a Path Animation / パスアニメーションの作成

Step 1: Create a Path / ステップ1：パスを作成

1. Add → Curve → Bezier Curve
   Add → Curve → Bezier Curve

2. Enter Edit Mode (Tab)
   編集モードに入る（Tab）

3. Select control points and move (G) to shape path
   コントロールポイントを選択して移動（G）してパスを形成

4. Can add more points: Extrude (E)
   さらにポイントを追加可能：押し出し（E）

Step 2: Create Object to Animate / ステップ2：アニメーションするオブジェクトを作成

  • Add simple object: Shift+A → Mesh → UV Sphere (or any object)
  • Position at start of path

Step 3: Add Follow Path Constraint / ステップ3：Follow Path制約を追加

1. Select the object (sphere)
   オブジェクト（球体）を選択

2. Properties panel → Constraints (chain link icon)
   プロパティパネル → Constraints（チェーンリンクアイコン）

3. Add Constraint → Relationship → Follow Path
   制約を追加 → Relationship → Follow Path

4. Target: Click eyedropper and select the curve
   ターゲット：スポイトをクリックして曲線を選択

5. Check "Follow Curve" option
   「Follow Curve」オプションにチェック

Step 4: Animate Along Path / ステップ4：パスに沿ってアニメーション

1. Select the curve (not the object)
   曲線を選択（オブジェクトではない）

2. Object Data Properties → Path Animation
   Object Data Properties → Path Animation

3. Set Frames: e.g., 0 to 120 (4 seconds at 30fps)
   フレームを設定：例：0から120（30fpsで4秒）

4. Press Spacebar to play animation
   スペースバーを押してアニメーションを再生

Timeline Controls / タイムラインコントロール

🎬 Animation Shortcuts / アニメーションショートカット:

  • Spacebar = Play/Pause animation / アニメーションを再生/一時停止
  • Left/Right Arrow = Step forward/backward one frame / 1フレーム前後に移動
  • Shift+Left Arrow = Jump to first frame / 最初のフレームにジャンプ
  • Shift+Right Arrow = Jump to last frame / 最後のフレームにジャンプ
  • Timeline: Bottom of screen, shows frame range
  • Playhead: Blue vertical line, shows current frame
  • Scrubbing: Drag playhead to preview animation

Segment 6: Introduction to Bone Rigging

What is Rigging? / リギングとは何か？

English: Rigging is creating a skeleton system (bones/armature) that controls how a mesh deforms. Essential for character animation and complex mechanical movements.

日本語: リギングは、メッシュがどのように変形するかを制御する骨格システム（ボーン/アーマチュア）を作成することです。キャラクターアニメーションと複雑な機械的動きに不可欠です。

Basic Armature / 基本的なアーマチュア

1. Add Armature / アーマチュアを追加

Shift+A → Armature → Single Bone
Shift+A → Armature → Single Bone

2. Edit Armature in Edit Mode / 編集モードでアーマチュアを編集

  • With armature selected: Tab → Edit Mode
  • Select bone head or tail and move (G)
  • E to extrude and add connected bones

3. Parent Mesh to Armature (Skinning) / メッシュをアーマチュアに親子付け（スキニング）

1. Select mesh object
   メッシュオブジェクトを選択

2. Shift+Click to also select armature (armature should be bright orange)
   Shift+クリックでアーマチュアも選択（アーマチュアは明るいオレンジ色になるはず）

3. Ctrl+P → With Automatic Weights
   Ctrl+P → With Automatic Weights

4. Now bones control mesh
   これでボーンがメッシュを制御

4. Pose Mode / ポーズモード

  • Select armature
  • Change to Pose Mode (dropdown at top left, or Ctrl+Tab)
  • Select individual bones
  • Rotate (R) or move (G) bones
  • Mesh deforms with bone movements

📌 Important Note:

English: Rigging is complex for beginners. This is just an introduction. We'll use pre-rigged characters later in the course. For now, just observe the demonstration.

日本語: リギングは初心者には複雑です。これは単なる入門です。コースの後半でリグ済みキャラクターを使用します。今のところ、デモを観察するだけです。

Export Settings / エクスポート設定

English: When you're ready to use your Blender models in Unity, you'll export them as FBX files.

日本語: BlenderモデルをUnityで使用する準備ができたら、FBXファイルとしてエクスポートします。

Export Process / エクスポートプロセス:
1. File → Export → FBX (.fbx)
   File → Export → FBX（.fbx）

2. Settings:
   設定:
   - Selected Objects (if exporting specific items)
   - Selected Objects（特定のアイテムをエクスポートする場合）

   - モディファイアを適用

Common Pitfalls / よくある落とし穴

1. Materials Not Showing / マテリアルが表示されない

Problem: Material is black or not visible / マテリアルが黒または見えない

Solution: Check viewport shading mode - use Material Preview (3rd sphere) or Rendered view (4th sphere)

2. Texture Appears Stretched or Wrong / テクスチャが引き伸ばされるまたは間違っている

Problem: UV mapping not done correctly / UVマッピングが正しく行われていない

Solution: Re-unwrap with proper seams. Check UV layout in UV Editing workspace.

3. Too Many Seams / シームが多すぎる

Problem: UV layout is fragmented / UVレイアウトが断片化されている

Solution: Use fewer seams, only where necessary (sharp corners, natural breaks)

4. Texture File Not Found / テクスチャファイルが見つからない

Problem: Red/pink "missing texture" appearance / 赤/ピンクの「欠けテクスチャ」の見た目

Solution: Keep texture images in project folder, use File → External Data → Pack Resources

5. Animation Not Playing / アニメーションが再生されない

Problem: Timeline doesn't show movement / タイムラインが動きを表示しない

Solution: Check frame range (start/end), ensure Follow Path constraint is properly set with correct target

6. Bones Don't Affect Mesh / ボーンがメッシュに影響しない

Problem: Mesh doesn't move with bones / メッシュがボーンと一緒に動かない

Solution: Check parenting - select mesh, then armature, then Ctrl+P → With Automatic Weights
