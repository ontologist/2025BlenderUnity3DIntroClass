Learning Objectives / 学習目標

English: By the end of this lecture, students will be able to:

  • Understand materials and shaders in Blender
  • Apply colors and materials to 3D objects
  • Perform basic UV unwrapping
  • Apply image textures to models
  • Create simple path-based animations
  • Understand basic bone rigging concepts

日本語:

  • Blenderのマテリアルとシェーダーを理解する
  • 3Dオブジェクトに色とマテリアルを適用する
  • 基本的なUV展開を実行する
  • モデルに画像テクスチャを適用する
  • シンプルなパスベースのアニメーションを作成する
  • 基本的なボーンリギングの概念を理解する
