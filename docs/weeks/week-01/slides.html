🎨 What is 3D Modeling? / 3Dモデリングとは？

3D Graphics Fundamentals

3D Modeling is the process of creating three-dimensional objects in digital space.

Basic Components:

  • Vertices: Points in 3D space
  • Edges: Lines connecting vertices
  • Faces: Surfaces defined by edges

Coordinate System:

  • X-axis: Left/Right (Red)
  • Y-axis: Forward/Back (Green)
  • Z-axis: Up/Down (Blue)

3Dグラフィックスの基礎

3Dモデリングは、デジタル空間で3次元オブジェクトを作成するプロセスです。

基本コンポーネント:

  • 頂点: 3D空間の点
  • エッジ: 頂点を結ぶ線
  • 面: エッジによって定義される表面

座標系:

  • X軸: 左/右（赤）
  • Y軸: 前/後（緑）
  • Z軸: 上/下（青）

🖥️ Blender Interface Overview

Main Interface Areas:

  1. 3D Viewport (Center)
    • Main working area for modeling
  2. Outliner (Top Right)
    • Scene hierarchy and object list
  3. Properties Panel (Bottom Right)
    • Object properties and settings
  4. Timeline (Bottom)
    • Animation controls (used later)
  5. Tool Shelf (Left)
    • Modeling tools and operations

主要なインターフェースエリア:

  1. 3Dビューポート（中央）
    • モデリングのメイン作業エリア
  2. アウトライナー（右上）
    • シーン階層とオブジェクトリスト
  3. プロパティパネル（右下）
    • オブジェクトのプロパティと設定
  4. タイムライン（下部）
    • アニメーションコントロール（後で使用）
  5. ツールシェルフ（左側）
    • モデリングツールと操作

🔧 Transform Tools in Detail

Move (G - Grab)

  • G → Move freely
  • G → X/Y/Z → Constrain to axis
  • G → Shift + X/Y/Z → Exclude axis
  • G → Type number → Move exact distance

Rotate (R)

  • R → Rotate freely
  • R → X/Y/Z → Rotate around axis
  • R → Type number (e.g., 90) → Rotate exact degrees

Scale (S)

  • S → Scale uniformly
  • S → X/Y/Z → Scale along axis
  • S → Type number → Scale to exact size

移動 (G - Grab)

  • G → 自由に移動
  • G → X/Y/Z → 軸に制約
  • G → Shift + X/Y/Z → 軸を除外
  • G → 数字を入力 → 正確な距離で移動

回転 (R)

  • R → 自由に回転
  • R → X/Y/Z → 軸周りに回転
  • R → 数字を入力（例：90） → 正確な角度で回転

スケール (S)

  • S → 均一にスケール
  • S → X/Y/Z → 軸に沿ってスケール
  • S → 数字を入力 → 正確なサイズにスケール

🔄 Object Mode vs Edit Mode

Object Mode / オブジェクトモード

English:

  • Work with entire objects
  • Move, rotate, scale whole objects
  • Add/delete objects
  • Duplicate objects

日本語:

  • オブジェクト全体で作業
  • オブジェクト全体を移動、回転、スケール
  • オブジェクトを追加/削除
  • オブジェクトを複製

Edit Mode / 編集モード

English:

  • Modify object geometry
  • Select vertices, edges, faces
  • Extrude, inset, loop cut
  • Shape and sculpt objects

日本語:

  • オブジェクトのジオメトリを変更
  • 頂点、エッジ、面を選択
  • 押し出し、インセット、ループカット
  • オブジェクトを形成してスカルプト

Toggle with Tab key / Tabキーで切替

✨ Edit Mode Selection Modes

Selection Modes:

1 Vertex Mode Select individual points
2 Edge Mode Select edges between vertices
3 Face Mode Select surfaces/polygons

Selection Tools:

  • Left Click - Select single element
  • Shift + Left Click - Add to selection
  • A - Select all
  • Alt + A - Deselect all
  • B - Box select
  • C - Circle select

選択モード:

1 頂点モード 個々の点を選択
2 エッジモード 頂点間のエッジを選択
3 面モード 表面/ポリゴンを選択

選択ツール:

  • 左クリック - 単一要素を選択
  • Shift + 左クリック - 選択に追加
  • A - すべて選択
  • Alt + A - すべて選択解除
  • B - ボックス選択
  • C - サークル選択

🔨 Essential Edit Mode Tools

Tool / ツール Shortcut Description / 説明
Extrude / 押し出し E Create new geometry by extending selected faces / 選択した面を拡張して新しいジオメトリを作成
Inset Faces / インセット面 I Create inset within selected faces / 選択した面内にインセットを作成
Loop Cut / ループカット Ctrl + R Add edge loops to mesh / メッシュにエッジループを追加

➕ Adding Primitive Shapes / プリミティブ形状の追加

Press Shift + A to open Add Menu

Common Primitives:

  • Cube: 6-sided box (starting object)
  • UV Sphere: Smooth sphere (32 segments)
  • Cylinder: Tube shape with circular ends
  • Cone: Pointed shape narrowing to a point
  • Plane: Flat square surface
  • Torus: Donut shape

Pro Tip:

After adding object, you can immediately adjust settings in the bottom-left corner!

Shift + Aを押して追加メニューを開く

一般的なプリミティブ:

  • 立方体: 6面のボックス（開始オブジェクト）
  • UV球: 滑らかな球（32セグメント）
  • 円柱: 円形の端を持つチューブ形状
  • 円錐: 点に向かって狭くなる尖った形状
  • 平面: 平らな正方形の表面
  • トーラス: ドーナツ形状

プロのヒント:

オブジェクトを追加した後、左下隅で設定をすぐに調整できます！

🏠 Demo Project: Simple House

Step-by-Step Instructions:
  1. Base of House / 家の基礎:
    • Start with default cube
    • Press S → type 2 → Enter (scale up)
  2. Roof Base / 屋根ベース:
    • Shift + A → Mesh → Cube
    • G → Z → type 2 → Enter (move up)
    • S → X → 1.2 → Enter (scale X)
    • S → Z → 0.2 → Enter (flatten)
  3. Roof / 屋根:
    • Shift + A → Mesh → Cone
    • G → Z → position above roof base
    • S → scale to fit
    • R → Y → 90 → Enter (if needed)

📦 GitHub Setup for Coursework

Repository Setup:

  1. Create new repository: game-dev-2025-yourname
  2. Initialize with README
  3. Add .gitignore template for Unity
  4. Clone repository to local machine

Folder Structure:

game-dev-2025-yourname/ ├── week-01/ ├── week-02/ ├── week-03/ ... └── week-12/

Basic Workflow:

  1. Save Blender file in appropriate week folder
  2. Commit changes with descriptive message
  3. Push to GitHub

リポジトリセットアップ:

  1. 新しいリポジトリを作成: game-dev-2025-yourname
  2. READMEで初期化
  3. Unity用の.gitignoreテンプレートを追加
  4. ローカルマシンにリポジトリをクローン

フォルダ構造:

game-dev-2025-yourname/ ├── week-01/ ├── week-02/ ├── week-03/ ... └── week-12/

基本的なワークフロー:

📝 Week 1 Assignment / 第1週の課題

Assignment: Create a Simple Maze Layout

課題：シンプルな迷路レイアウトを作成

Requirements:

  • Create a maze using cubes as walls

  • Include:

    • 床平面

  • GitHubにプッシュ

⚠️ Common Mistakes to Avoid

Navigation Issues:

  • ❌ Forgetting middle mouse button alternatives
  • ✅ Use Alt+Left Click if no MMB

Lost in 3D Space:

  • ❌ Can't find objects
  • ✅ Press Home or NumPad . to frame all

Mode Confusion:

  • ❌ Not knowing current mode
  • ✅ Check mode indicator (top-left)

File Management:

  • ❌ Not saving or wrong location
  • ✅ Save early, save often
  • ✅ Use proper folder structure

ナビゲーションの問題:

  • ❌ 中ボタンの代替を忘れる
  • ✅ MMBがない場合はAlt+左クリックを使用

3D空間で迷子:

  • ❌ オブジェクトが見つからない
  • ✅ HomeまたはNumPad .を押してすべてをフレーム

モードの混乱:

  • ❌ 現在のモードがわからない
  • ✅ モードインジケータを確認（左上）

ファイル管理:

  • ❌ 保存しない、または間違った場所に保存
  • ✅ 早めに保存、頻繁に保存
  • ✅ 適切なフォルダ構造を使用
