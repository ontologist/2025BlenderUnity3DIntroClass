Learning Objectives / 学習目標

English: By the end of this lecture, students will be able to:

  • Build games for multiple platforms (Windows, Mac, WebGL)
  • Configure build settings for different platforms
  • Create professional game branding assets (icon, splash screen)
  • Optimize builds for performance and file size
  • Publish to distribution platforms (Itch.io, GitHub Releases)
  • Create game documentation and press kits
  • Understand monetization and distribution options
  • Set up game analytics and metrics

日本語: この講義の終了までに、学生は以下ができるようになります：

  • 複数のプラットフォーム（Windows、Mac、WebGL）向けにゲームをビルド
  • 異なるプラットフォームのビルド設定を構成
  • プロフェッショナルなゲームブランディングアセット（アイコン、スプラッシュスクリーン）を作成
  • パフォーマンスとファイルサイズのためにビルドを最適化
  • 配布プラットフォーム（Itch.io、GitHub Releases）に公開
  • ゲームドキュメントとプレスキットを作成
  • 収益化と配布オプションを理解
  • ゲームアナリティクスとメトリクスをセットアップ
