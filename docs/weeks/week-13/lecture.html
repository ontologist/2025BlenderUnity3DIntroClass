Learning Objectives / 学習目標

English: By the end of this lecture, students will be able to:

  • Implement advanced game features (collectibles, power-ups, multiple levels)
  • Create save/load system using PlayerPrefs and JSON
  • Build a comprehensive settings menu with volume and graphics controls
  • Optimize game performance using Unity Profiler
  • Implement object pooling for better memory management
  • Optimize assets for better performance
  • Apply professional optimization techniques
  • Add analytics and debugging tools

日本語: この講義の終了までに、学生は以下ができるようになります：

  • 高度なゲーム機能を実装（収集アイテム、パワーアップ、複数レベル）
  • PlayerPrefsとJSONを使用してセーブ/ロードシステムを作成
  • ボリュームとグラフィックスコントロールを含む包括的な設定メニューを構築
  • Unity Profilerを使用してゲームパフォーマンスを最適化
  • メモリ管理改善のためのオブジェクトプーリングを実装
  • パフォーマンス向上のためのアセットを最適化
  • プロフェッショナルな最適化技術を適用
  • アナリティクスとデバッグツールを追加
