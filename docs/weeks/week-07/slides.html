What is AI in Games?

  • AI (Artificial Intelligence) makes NPCs act intelligently
  • Enemies that chase, patrol, or avoid obstacles
  • Creates challenging and dynamic gameplay

NavMesh (Navigation Mesh):

  • Pre-calculated walkable areas in the scene
  • Enables pathfinding for AI agents
  • Handles obstacle avoidance automatically
  • More efficient than raycast-based navigation

When to Use NavMesh:

  • ‚úÖ Ground-based navigation
  • ‚úÖ Static obstacles
  • ‚úÖ Multiple AI agents

Enhanced EnemyController with Detection Range

public float detectionRange = 10f;
public float chaseRange = 15f;

void FixedUpdate()
{
    if (player != null)
    {
        float distance = Vector3.Distance(
            transform.position, player.position);

        if (distance <= detectionRange)
        {
            // Chase player
            agent.SetDestination(player.position);
        }
        else if (distance > chaseRange)
        {
            // Stop chasing
            agent.ResetPath();
        }
    }
}

Benefits:

  • Enemy only chases when player is nearby
  • Performance optimization (fewer calculations)
  • Better gameplay feel
  • Allows for stealth gameplay
