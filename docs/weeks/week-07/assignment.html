🏆 Midterm Project / 中間プロジェクト

Due Date: Before Week 8 lecture / 第8週講義前

Points: 25 points / 25点

Estimated Time: 6-8 hours / 6-8時間

Submission: Complete Unity project + executable build

Assignment Description / 課題概要

English: Complete and polish your ball maze game with multiple enemies, enhanced UI, game state management, and visual effects. This is your midterm project submission demonstrating all skills learned in Weeks 1-7.

日本語: 複数の敵、強化されたUI、ゲーム状態管理、視覚効果を備えたボール迷路ゲームを完成させ、仕上げます。これは、第1-7週で学んだすべてのスキルを示す中間プロジェクト提出です。

Requirements / 要件

1. Multiple AI Enemies 5 points

  • Implement at least 3 AI enemies in the game
  • Enemies spawn at different locations
  • All enemies use NavMesh navigation
  • Enemies don't interfere with each other
  • Performance remains smooth

2. Enhanced Game State Management 5 points

  • Complete GameManager with all states: Playing, Won, Lost, Paused
  • State transitions work correctly
  • Controls disabled in wrong states
  • UI updates based on current state
  • Restart functionality works

3. Polished UI System 5 points

  • Enhanced win/lose panels
  • Score or timer display
  • Clear restart buttons
  • UI animations (fade in/out)
  • Professional appearance

4. Visual Polish 5 points

  • Particle effects for key events (win, lose, collisions)
  • Visual feedback for player actions
  • Screen effects (fade transitions)
  • Improved lighting
  • Overall visual appeal

5. Game Balancing & Testing 5 points

  • Game feels challenging but fair
  • Difficulty is appropriate
  • No game-breaking bugs
  • Smooth performance
  • Complete playtesting documentation

Bonus Points / ボーナス点 (+5 points max)

1. Advanced Features (+2 points)

Multiple levels, collectible items system, power-ups, or time attack mode.

2. Polish & Effects (+2 points)

Multiple particle effects, screen shake, post-processing effects, sound effects integration.

3. Documentation (+1 point)

Comprehensive README, code documentation, design document, playtesting report.

Submission Checklist / 提出チェックリスト

  • Complete Unity project with all scripts
  • Working executable build
  • All features from previous weeks working
  • GitHub repository organized and committed
  • Screenshots/videos of gameplay
  • README with instructions
  • Project builds without errors
