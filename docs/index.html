📚 Duration

100 min/lecture

🎯 Target Audience

Complete beginners

🛠️ Tools

Blender 4.0+

Unity3D 2022 LTS

GitHub & Cursor

What You'll Learn / 学習内容

  • Blender Basics (Weeks 1-2): 3D modeling, materials, textures, and animations
  • Unity3D Fundamentals (Weeks 3-4): Game engine basics, physics, and scripting
  • Game Development (Weeks 5-9): Build a complete maze game with AI enemies
  • Polish & Audio (Weeks 10-11): Add sound effects and background music
  • Final Project (Week 12): Complete and present your game
